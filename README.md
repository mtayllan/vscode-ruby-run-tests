# Ruby Run Tests

Quick commands to run your ruby tests

## Features

- Toggle between program file and test file (rspec and minitest)
- Run test for current program file or test file

## Keybindings

- Toggle: `alt+d`
- Run test: `alt+f`
- Run test for current line: `alt+v`
